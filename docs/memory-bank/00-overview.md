# 00 - Overview

RBMC monitor web application for Brazilian GNSS/RBMC station visibility.

## Goals
- Show station status and location on map and list views.
- Prioritize real-time NTRIP source with IBGE fallback.
- Provide clear operator UI with theme/language support.
- Keep data source attribution visible.

## Scope
- Backend: station aggregation and fallback logic.
- Frontend: map visualization, station search/filter, policy/documentation panel.
