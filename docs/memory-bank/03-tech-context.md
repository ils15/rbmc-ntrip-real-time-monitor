# 03 - Tech Context

## Stack
- Backend: Node.js + Express
- Frontend: React + Vite + React-Leaflet
- Styling: CSS with theme variables

## Main Commands
- `npm run dev:all`
- `npm run build:client`
- `npm test`

## Runtime Notes
- Frontend expects backend on local API routes.
- Theme and language preferences are stored in browser local storage.
