# 02 - Components

## Backend
- `server.js`: API endpoints and HTTP server.
- `src/ibge-client.js`: station retrieval, cache usage, source fallback.

## Frontend
- `client/src/App.jsx`: orchestration, search/filter, views, source metadata.
- `client/src/components/StationMap.jsx`: Leaflet map, markers, popups/tooltips.
- `client/src/components/StationList.jsx`: card-based station listing.
- `client/src/components/PolicyPanel.jsx`: privacy/reference documentation tab.
- `client/src/i18n.jsx`: PT/EN translation dictionary.
- `client/src/App.css`: theme and component styling.
