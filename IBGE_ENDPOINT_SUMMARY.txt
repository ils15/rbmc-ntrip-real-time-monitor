⚙️ CACHE STRATEGY:

Default TTL: 1 hour (3600000 ms)
Strategies:
  1. Redis (if REDIS_URL env var set) - Shared cache
  2. In-memory fallback - Development/testing
  3. NTRIP fallback - If IBGE API fails

Performance:
  - Cached: ~45-100ms
  - Fallback: <200ms (mock data)
  - First fetch: ~5 seconds (timeout limit)
